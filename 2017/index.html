PROGRAMAÇÃO #LARACONFBRASIL

Dia 17 | Dia 18

08:00 CREDENCIAMENTO
08:00 COFFEE
09:00 Camila Silva Fernandes API RESTful com Laravel

O Laravel é um Framework, com sintaxe elegante e expressiva, que tem como principal característica ajudar a desenvolver aplicações seguras e robustas de forma ágil e eficiente, com uma documentação vasta e acessível, tornando fácil e rápido o desenvolvimento. Vamos aproveitar a agilidade para criação de rotas controlers e views que o Laravel nos proporciona para construir uma API RESTFul simples.
10:00 Gabriel Caruso Collections, uma solução?!

Collections do Laravel seria uma “gambiarra” para o trabalho com arrays, strings, e outros dados, seja facilitado? Você sabe o que é uma Collection do Laravel? Quais benefícios ela te trás ao seus projetos Laravel, e PHP?
11:00 Ravan Scafi Entenda de uma vez por todas o Container do Laravel

Mais do que uma forma de injetar dependências no construtor de classes, o container do Laravel é uma ferramenta poderosa do framework, que você já usa e talvez não conheça. Entenda como o Container auxilia no Dependency Inversion Principle do SOLID, conheça os principais tipos de bindings e, de quebra, aprenda sobre Contracts, Service Providers e Facades.
11:30 LUNCH TIME!
13:00 Marcos Felipe Nalin Furtado O mundo além do SOLID e dos Design Patterns

Adotar um Design Pattern e falar que SOLID deve ser usado é fácil, porém isso tem um efeito colateral que pode ser mais oneroso que benéfico, devemos saber e ver o mundo além da aplicação dos Design Patterns e dos princípios S.O.L.I.D.
13:30 Leandro Costa Martins Provisionando seu ambiente Laravel com Ansible

Tenha seu ambiente Laravel provisionado em apenas alguns minutos utilizando o Ansible. Conheça essa ferramenta simples, poderosa e open-source, que ajuda você a provisionar e configurar a infraestrutura necessária para sua aplicação.
14:00 Gabriel Koerich Workflow & Infra para aplicações web

A ideia é mostrar o workflow desde o desenvolvimento até o deploy em produção, passando pelo software para gerenciamento de projetos, GitHub & pull requests, config, testes automatizados, deploy contínuo com zero downtime, monitoring e toda a configuração de infra como CDN, multi servers, load balancer, master/slave database, queues e como tudo isso se encaixa no ambiente do Laravel.
15:00 COFFEE
15:30 Vitor Talaia Aplicações CLI com Crystal

Nessa talk, veremos como criar CLIs usando a linguagem de programação Crystal. Uma linguagem relativamente nova que tem como objetivo ser rápida como C e agradável como Ruby.
16:00 Lucas Pires Mattos Criando pacotes com Laravel

Aprenda como trabalhar com Service Providers, Routing, Controllers, Views, Migrations e muito mais ao criar seus próprios pacotes.
17:00 Ricardo Plansky Jr. Como construir um MVP com design de código evolutivo guiado a testes

O termo MVP está cada vez mais evidente no dia-a-dia das empresas e desenvolvedores, muito pelo time to market ou até mesmo porque empresas que não têm a filosofia de qualidade que e acham que entregar um MVP sempre é saudável e com menor custo. Mas por outro lado, temos um conceito que MVP é um código temporário e que não precisamos focar tanto em qualidade ou testes, o que não é verdade sempre. Nesta talk vou falar como evitar de ter um código legado construindo um MVP com uma suíte de testes segura e com um design de código preparado para crescer utilizando Lumen.
08:00 CREDENCIAMENTO
08:00 COFFEE
09:00 Fabio Vedovelli Apenas Laravel não basta!

A palestra discorrerá sobre a importância do front end, UX, suas complexidades e como o Laravel torna a curva de aprendizado e o próprio desenvolvimento front end mais suaves para o desenvolvedor.
09:30 Erik Vanderlei Fernandes Laravel e seu ecossistema

Como o ecossistema Laravel possibilita um desenvolvimento mais rápido.
10:30 Guilherme Guitte Testes automatizados: Tomando as rédeas do seu código (ou de outros)

Por que (deveríamos) escrevemos testes automatizados? Como começar? Qual o benefício para sua carreira? Minha aplicação é legada, como começar? O que é pirâmide de testes? Iceberg?
11:30 LUNCH TIME!
13:00 Vinícius Bail Alonso Clean Code: como tornar nossos códigos legiveis para seres humanos

Escrever código para um compilador é fácil, basta seguir as regras de sintaxe. Difícil é tornar esse código legível para quem irá trabalhar com ele no futuro. Nessa palestra vamos ver pequenos detalhes como, nomes expressivos e funções bem escritas, podem transformar um software.
13:30 Marcel dos Santos Melhore a Qualidade do seu Código com Object Calisthenics

Um código com qualidade é essencial para um projeto de sucesso e existem diversas práticas que ajudam a alcançar esse objetivo. Object Calisthenics é uma série de boas práticas que ajudarão a tornar o seu legível, manutenível, testável e compreensível. Com uma dose pequena de exercícios diários, você notará a diferença no seu código.
14:30 Wilton Guilherme Produtividade no Desenvolvimento usando Laravel com Docker

Mostrar como um ambiente de desenvolvimento padronizado pode aumentar a produtividade das equipes evitando retrabalhos e conflitos com times de operação na hora de realizar deploy em produção.
15:00 COFFEE
15:30 Eduardo Cesar Porque escolhemos o Laravel - uma visão que está além do ponto de vista técnico

Adotar um framework em um ambiente corporativo é um desafio que envolve muito mais que questões técnicas, o peopleware que está em volta desse processo é peça fundamental. Nesta apresentação serão levantados pontos que foram relevantes para adoção de um framework e como o Laravel se encaixou bem dentro desse cenário.
16:00 Vinicius Reis Domínios, do monolítico ao micro-serviço

Arquitetura de de software vai além de organização de classes e arquivos. Com sistemas mais complexos e grandes demandas para se adaptar e evoluir rapidamente, pensar em domínios e micro-serviços é importante para se conseguir bons resultados. O objetivo aqui é explorar esta dinâmica em suas várias formas.
17:00 Diego Hernandes Laravel: O Estado da Arte

Conheça a trajetória, conceitos, filosofia e comunidades que transformaram uma pequena biblioteca PHP em um das ferramentas de desenvolvimento mais utilizados do mundo! Uma reflexão sobre o passado, presente e futuro do desenvolvimento de software do ponto de vista humanista.
